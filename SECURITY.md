# Security policy

Cogwild находится на ранней стадии разработки и пока не имеет публичных стабильных релизов.

## Сообщение об уязвимости

Не публикуйте сведения об уязвимости, позволяющей выполнить чужой код, повредить мир или получить доступ к данным сервера, в открытом Issue.

Свяжитесь с владельцем репозитория через профиль GitHub `@Deenfoool` и укажите:

- версию Cogwild;
- версию Minecraft и Forge;
- способ воспроизведения;
- ожидаемое и фактическое поведение;
- лог или минимальный пример;
- возможные последствия.

Обычные игровые ошибки, сбои моделей, рецептов и баланса можно публиковать через шаблон Bug report.

## Поддерживаемые версии

До первого релиза поддерживается только ветка `develop`. После появления релизов таблица поддерживаемых версий будет обновлена.
