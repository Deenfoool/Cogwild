# Changelog

Все заметные изменения Cogwild будут фиксироваться в этом файле.

## [0.1.0-dev] — в разработке

### Added

- базовый проект Minecraft Forge 1.21.1;
- Java 21 toolchain;
- зависимость GeckoLib;
- главный класс мода;
- регистрация предметов, блоков и творческой вкладки;
- предмет `cogwild:steel_ingot`;
- блок `cogwild:machine_casing`;
- русская и английская локализации;
- рецепт и loot table корпуса машины;
- Gradle Wrapper;
- GitHub Actions для автоматической сборки;
- документация архитектуры и именования;
- шаблоны Pull Request и Issues;
- правила участия и лицензия проекта.

### Notes

Модели первых объектов временно используют ванильные текстуры. Версия ещё не предназначена для обычной игры.
